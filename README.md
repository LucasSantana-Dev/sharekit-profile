# Sharekit Profile: lucassantana

A clean, reusable Claude Code profile demonstrating coding fundamentals and helpful behavioral skills.

## Install

```bash
npx @lucassantana/sharekit install lucassantana
```

This mirrors into:
- `~/.claude/CLAUDE.md` — core rules (think before code, commit discipline, code standards)
- `~/.claude/skills/caveman/` — ultra-compressed communication mode
- `~/.claude/skills/ponytail/` — lazy/minimal design philosophy

## What's Inside

### CLAUDE.md
A fresh, generic guide to coding with intent. Covers:
- Think before you code: understand, validate, ship incrementally
- Commit discipline: atomic commits, clear messages, clean history
- Code standards: readability, consistency, focused testing
- PR and review hygiene
- Debugging mindset

This is a starting point — adapt it to your team's style.

### Caveman
Ultra-compressed output mode: drop articles, filler, pleasantries; keep technical substance. ~75% token savings. Activate with `/caveman`.

### Ponytail
Lazy senior developer mode: YAGNI, stdlib first, native features before dependencies, one line before fifty. Activate with `/ponytail`.
